# Tapp

A fast-paced, real-time battle between two players. Each player taps their side of the screen to push the color boundary in their favor — the first to cover the entire screen wins.

## How to play

1. Enter your name and a room ID, then hit **Join**.
2. Share the room ID with your opponent so they can join the same room.
3. Once both players are in, the game starts — tap as fast as you can to claim the screen.
4. The screen is divided by color. Every tap pushes the boundary toward your opponent. Cover the full screen to win.
5. After the match, either player can start a rematch.

## Contributing

1. Clone this repo and the [backend repo](https://github.com/theankurkedia/tapp-be).
2. Create an `.env` file with the contents of `.env.local`.
3. Install the dependencies in both repos.
4. Run the backend repo first, then this repo.
